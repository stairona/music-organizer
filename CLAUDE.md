# CLAUDE.md

## Project
Music Organizer

## Goal
Build a Python tool that scans a music folder and organizes files by genre.

## Tech Stack
- Python 3
- mutagen

## Rules
- Write clean, modular Python code
- Prefer readability over cleverness
- Add comments for important logic
- Handle errors safely
- Do not delete files automatically without confirmation

## Files to create
- main.py
- requirements.txt
- README.md

## Expected behavior
The program should:
1. Scan a target music directory
2. Detect genre metadata from files
3. Create folders by genre
4. Move or copy files into the correct genre folder
5. Print a summary of processed files
